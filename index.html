Welcome to the World of HTML and CSS

Introduction

What is HTML?

HTML is an abbreviation for Hyper Text Markup Language. It is the code that structures a webpage.

What is CSS?

CSS is an abbreviation for Cascading Style Sheet. It's the code that styles the appearance of the webpage.

HTML Basics

What is a tag?

A tag specifies a specific instruction on the page.

What is an element?

An element is everything from the start tag to the end tag.

What is an attribute?

An attribute allows for add ons to an element and is usually found in the opening tag.

CSS Basics

What is a selector?

A selector is a tag to reference the element that you want to style.

What is a class?

A class is an identifier that allows us to identifiy and style multiple elements.

What is an ID?

An ID works similar to a class but for specific elements.
